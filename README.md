# quran-rest-api
Rest Api untuk Web Al-Qur'an

## How to use
```
//GET LIST surah 1 - 114
https://quran-webapi.herokuapp.com

//GET satu surah spesifik
https://quran-webapi.herokuapp.com/surah/:id -> 1 - 114
```
